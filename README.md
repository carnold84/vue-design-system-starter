# Vue Design System Starter

## Project setup
```
yarn install
```

### Starts Storybook for development
```
yarn storybook
```

### Compiles and minifies for production
```
yarn build
```

### Run your unit tests
```
yarn test:unit
```

### Lints and fixes files
```
yarn lint
```

See [Configuration Reference](https://cli.vuejs.org/config/).
